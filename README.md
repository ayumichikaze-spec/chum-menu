# CHUM APARTMENT Menu Site

チャムアパートメントの多言語メニューサイト

## 構成
- `index.html` : メインメニュー(ランチ/ディナー切替)
- `drink.html` : ドリンクメニュー
- `takeout.html` : テイクアウトメニュー
- `campaign.html` : 期間限定キャンペーン

## 対応言語
日本語 / English / Français / 한국어 / 繁體中文 / 简体中文

## ホスティング
Cloudflare Pages (https://menu.chum-apt.net)

## 更新方法
このリポジトリのHTMLファイルを編集してcommitすれば、Cloudflare Pages が自動デプロイ。
翻訳の追加・修正は `i18n/translations.json` を編集。
